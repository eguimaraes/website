O que é?
O Microsoft Project Server2013 é uma solução online flexível para o gerenciamento de portfólio de projetos (PPM) e para o trabalho diário. Os membros da equipe, os participantes de projetos e os tomadores decisões podem começar a trabalhar, priorizar investimentos em portfólio de projetos e produzir o valor de negócios pretendido de praticamente qualquer lugar.
Saiba mais em: https://products.office.com/pt-br/project/enterprise-project-server

Como posso aplicar em meu trabalho?
Economize milhares de reais em horas de desenvolvimento pois o Project Server é a ferramenta ideal para o rápido desenvolvimento de portais de PMO e acompanhamento de projetos. Com o Project Server uma pequena equipe de colaboradores poderá desenvolver rapidamente sofisticados portais e aplicações.Neste curso você aprenderá a:
  • Publicar e acompanhar seus projetos na Web
  • Desenvolver páginas Web com o SharePoint
  • Desenvolver sites de Wiki Corporativa (Como a Wikipedia)
  • Desenvolver listas e Bibliotecas de Documentos
  • criar rapidamente um GED
  • Desenvolver Fluxos de Trabalho
  • Publicar Planilhas e relatórios criados no Excel na Web
